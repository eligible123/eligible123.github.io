# 百度Java一面核心要点

## 一、Java基础

**集合框架**：ArrayList基于动态数组，查询O(1)，增删O(n)；LinkedList基于双向链表，增删O(1)，查询O(n)。HashMap底层是数组+链表+红黑树（JDK8），默认负载因子0.75，扩容为2倍。

**多线程**：synchronized基于Monitor实现，JDK6后有偏向锁→轻量级锁→重量级锁升级；volatile保证可见性和有序性，不保证原子性。线程池核心参数：corePoolSize、maximumPoolSize、keepAliveTime、workQueue、handler。

**JVM**：堆分新生代（Eden+Survivor）和老年代；GC算法：标记-清除、复制、标记-整理；CMS和G1区别在于G1可预测停顿时间。

## 二、数据库

**MySQL**：InnoDB支持事务和行锁，索引采用B+树；事务隔离级别默认RR，通过MVCC+间隙锁解决幻读；慢SQL优化：加索引、避免SELECT *、分页优化。

**Redis**：常用数据结构String/Hash/List/Set/ZSet；持久化方式RDB+AOF；缓存穿透用布隆过滤器，缓存雪崩用随机过期时间。

## 三、框架与中间件

**Spring**：IoC容器管理Bean生命周期；AOP基于动态代理（JDK/CGLIB）；@Transactional失效场景：非public方法、自调用、异常被catch。

**分布式**：CAP理论，Base理论；分布式锁实现：Redis SETNX、Zookeeper临时顺序节点。

## 四、项目经验

建议准备：技术选型原因、核心难点、性能优化措施、线上问题排查经验。回答时用STAR法则，突出个人贡献和技术深度。

## 5. Java百度日常实习一面

🟡 **中等** | 标签：Java, 百度, 实习, 后端, 面经

**Java百度日常实习一面要点总结**

**一、Java基础**
*   **核心特性**：理解面向对象、封装继承多态、平台无关性。
*   **关键类**：掌握`String`不可变性及`StringBuilder`/`StringBuffer`区别；熟悉`HashMap`的底层结构（数组+链表/红黑树）、扩容机制与线程不安全原因。
*   **高级特性**：了解`反射`、`泛型`、`异常处理`及常用设计模式。

**二、多线程与并发**
*   **线程基础**：掌握线程创建方式、生命周期与`sleep`/`wait`区别。
*   **并发编程**：理解`volatile`、`synchronized`、`Lock`的区别与应用场景；熟悉`线程池`的核心参数与工作流程；了解`CAS`原理与`AQS`框架。

**三、JVM**
*   **内存模型**：清晰划分`堆`、`栈`、`方法区`等区域的作用。
*   **垃圾回收**：了解常见GC算法（标记-清除、复制、标记-整理）与分代收集思想；熟悉`G1`收集器特点。
*   **调优基础**：掌握基本的JVM参数（如`-Xms`, `-Xmx`）和问题排查思路。

**四、数据库与中间件**
*   **MySQL**：掌握`索引`的类型与优化（B+树、最左匹配）；理解`事务`的ACID特性及隔离级别；熟悉`MVCC`机制。
*   **Redis**：了解常见数据结构、持久化方式（RDB/AOF）及缓存穿透、雪崩的解决方案。

**五、Spring框架**
*   **核心概念**：深刻理解`IoC`（控制反转）与`AOP`（面向切面编程）。
*   **关键机制**：掌握`Spring Bean`的生命周期、作用域；熟悉`Spring MVC`请求处理流程。
