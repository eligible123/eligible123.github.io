服务升级后数据库问题排查指南

发布于 2025-05-09 | 分类: 运维 | 标签: 数据库 升级 问题排查

服务升级后数据库问题

Oracle或者达梦数据库

Varchar/Nvarchar2->Text/Clob

达梦/Oracle数据库，不支持直接从Varchar/Nvarchar2转换到Text/Clob，因此这种转换会直接报错

处理方式为，新增字段A1定义类型，再从A复制数据至A1，删除旧字段A，将A1改名为A

oracle处理为： 
ALTER TABLE "h_system_setting" ADD ("paramValue1" CLOB);

alter table "h_system_setting" rename column "paramValue1" to "paramValue";
达梦处理为： 
ALTER TABLE "h_system_setting" ADD ("paramValue1" TEXT);

alter table "h_system_setting" rename column "paramValue1" to "paramValue";
