一、需求

1.流程从H3迁移到云枢的时候，采用的是导入导出的方式，且H3的的组织架构和云枢的组织架构没有做同步，导致每次迁移新的流程时候，需要手工去把原H3的流程涉及的工作组手工在云枢系统上对照重新配置一边，但是这种方式容易出现遗漏和错误，需要一个可以自动同步的工具。

2.流程做需求变更时候，需要去掉表单上的某些字段，但是原来的流程的分支条件和审批人配置上可能会有与该字段相关的配置，去掉了可能会导致流程审批出现问题，需要一个可以查询相关配置的工具。

二、思路

1.提供一个接口，将流程涉及到的的工作组从H3同步到云枢，在迁移新的流程的时候，就不用人工对比操作，这样既减少出错的概率也提高了工作效率。

2.提供一个接口，对比H3的流程和云枢的流程涉及到的工作组的差别，可以对上线的流程进行查漏补缺。

3.提供一个接口，将涉及到的字段的分支条件和审批人配置查询出来。

    • public Map queryRelateDataFieldByWorkflow(String workflowCode,String dataField){ Map result=new HashMap<>(); List> activities=new ArrayList<>(); List> conditions=new ArrayList<>(); result.put("activity",activities); result.put("condition",conditions); WorkflowTemplateModel template= getWorkflowManagementFacade().getLastWorkflowTemplate(workflowCode); List activityList= template.getActivities(); for(Activity act : activityList){ //传阅节点和用户节点 if(act.getActivityType()== ActivityType.PARTICIPANT||act.getActivityType()== ActivityType.CIRCULATE){ String participant = ""; if (act.getActivityType() == ActivityType.PARTICIPANT) { ParticipantActivity p = (ParticipantActivity) act; participant = p.getParticipant(); } else { CirculateActivity p = (CirculateActivity) act; participant = p.getParticipant(); } if(StringUtil.isNotEmpty(participant)&&participant.indexOf("{"+dataField+"}")>-1){ Map map=new HashMap<>(); map.put("code",act.getActivityCode()); map.put("name",act.getActivityName()); map.put("participant",participant); activities.add(map); } } } //分支条件 List rules= template.getRules(); for(Rule r:rules){ if(!r.isUtilizeElse()){ String text=r.getFormula()==null?"":r.getFormula(); if(text.indexOf("{"+dataField +"}")>-1){ Map map=new HashMap<>(); List preAct= activityList.stream().filter(s->s.getActivityCode().equals(r.getPreActivityCode())).collect(Collectors.toList()); List postAct= activityList.stream().filter(s->s.getActivityCode().equals(r.getPostActivityCode())).collect(Collectors.toList()); map.put("preActivityCode",r.getPreActivityCode()); map.put("preActivityName",preAct.get(0).getActivityName()); map.put("postActivityCode",r.getPostActivityCode()); map.put("postActivityName",postAct.get(0).getActivityName()); map.put("text",text); conditions.add(map); } } } return result;}
