树形视图-左树右表数据一一对应

一、项目背景

主线业务需要通过项目相关信息，构建一个树形结构的视图，并且需要实现对节点的增加、删除、移动等相关节点操作。

二、实现思路

1、引入ant-design vue 中 Tree 组件，根据后端返回结构，递归生成符合Tree组件的树形接口，完成左侧Tree布局

2、引入ant-design vue中 Table 组件，使用 table组件中的树形table的方法，通过slot生成右侧视图列表。

3、调整左右侧的每行的高度，保持一致。调整css样式，使表头固定在顶部

4、绑定ref,使用dom元素，计算左侧树形结构滚动的高度，赋值给右侧table滚动的高度，同理计算右侧滚动的高度，赋值给左侧Tree滚动的高度。

5、节点的删除、添加、上移、下移操作的原理：每次拿到操作节点的Node元素，用一个变量去接收它，或者用list集合去存储展开的父级节点。这个时候我们可以通过，需要变化位置节点的parentId,去匹配父节点，通过修改父节点的children属性就可以实现节点的添加、上移、下移的操作了。然后对整个tree解构。(注意:千万不用对节点进行深拷贝，那么节点的数据就无法正常改变了）。

三、实现步骤

1、引入ant-design vue中的Tree组件，递归生成Tree结构。

2、引入ant-design vue中 Table 组件

3、调整左右侧的每行的高度，保持一致。调整css样式，使表头固定在顶部

4、绑定ref,使用dom元素，计算滚动的高度

5、节点的删除、添加、上移、下移操作

四、效果展示

1、引入组件，调整样式，使左右两边一一对应。

2、固定顶部表头，计算滚动高度，赋值给需要滚动的组件，左右两边同步滚动。

3、添加节点，通过iframe,引入产品的详情页面，放到弹框中，当表单点击按钮的时候，通过标签页之间的消息传递，控制弹框的显示影藏。

4、降级操作，点击当前节点，获取同级节点信息，选择需要降级到的同级节点，调整数据，完成降级操作。
