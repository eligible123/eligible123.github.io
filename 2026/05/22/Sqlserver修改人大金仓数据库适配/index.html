Sqlserver修改人大金仓数据库适配

背景：

随着国产化进程的推进，越来越多项目将进行国产化改造，国产化数据库一般会选择人大金仓或者达梦数据库，下面将SQLserver改造为人大金仓数据库，简单列举一些不适配，需要改造的地方，方便后面的同事参考改造。

人大金仓不支持的常见语法：

正则搜索范围：.cs;.ascx;.asmx;.ashx;*.asax;

1、with(nolock) 不支持，

搜索：

删除

2、中括号不支持，

搜索：\[[^\"\'\(\)]*\]

删除

例如：select [State] from [dbo].[I_DocumentSend]

改为：select State from dbo.I_DocumentSend

3、语句中 +号不支持，

搜索：select [^\"]*\+

搜索：\+[^\"]*\'

搜索：\'[^\"]*\+

修改为 ||

比如： select (Name+'_'+State) as Name from dbo.I_DocumentSend

改为： select (Name ||'_'|| State) as Name from dbo.I_DocumentSend

4、关联Update语法

搜索：update .*

增加表名

例如：update a set a.ParentID=b.ObjectID from OT_User a inner join OT_OrganizationUnit b on a.ParentID=b.ObjectID

改为：update OT_User a set a.ParentID=b.ObjectID from OT_OrganizationUnit b where a.ParentID=b.ObjectID;

常用函数转换：

1、当前时间： getdate()

搜索：getdate\(\)

改为：NOW()

2、随机ID：newid()

搜索：newid\(\)

改为：SYS_GUID_NAME()

3、日期格式转换样例：CONVERT(varchar(100), getdate(), 23)

搜索：CONVERT\(.*\)

改为：to_char(NOW(),'yyyy-MM-dd')

-- 昨天

SELECT CURRENT_TIMESTAMP - 1

-- 明天

SELECT CURRENT_TIMESTAMP + 1

-- 30日后

SELECT CURRENT_TIMESTAMP + 30

4、判断字段长度条件 len(CarType)>0

搜索：len\(.*\)

改为：length(CarType)>0

5、转换保留2位小数：

搜索：CONVERT\(.*\)

例如：Convert(decimal(18,2),AggregateScoreCount/itemCount)

改为：round(AggregateScoreCount/itemCount,2)

6、格式转换

搜索：CONVERT\(.*\)

改为：to_number、to_char、to_date

7、like 模糊查询语句：like CONCAT('%', @InstanceName,'%')，

比如：

string sql = "SELECT * FROM Person WHERE I_DocumentSend LIKE CONCAT('%', @InstanceName,'%') ";

var Parameters=new SqlParameter[]{new SqlParameter("@InstanceName", "流程名称")};

8、时间比较： DATEDIFF(Month,Inspectiontime,getdate()) >=0

搜索：DATEDIFF\(

改为： DATEDIFF('mm',Inspectiontime,NOW())>=0，则 年份 Year 修改为'yy',月份Month 修改为'mm', 其他如下

年yy,yyyy

季度qq,q

月mm,m

年中的日dy,y

日dd,d

周wk,ww

星期dw,w

小时hh

分钟mi,n

秒ss,s

毫秒ms

微妙mcs

纳秒ns

9、列转行

搜索：for xml path

例如：,(STUFF((select ';' + WorkPlan from DB_SuperviseDepart where Pro_Id=A.Pro_Id for xml path('')),1,1,'')) as Name

改为：,(select string_agg(WorkPlan,';') from DB_SuperviseDepart where Pro_Id=A.Pro_Id ) as Name

9、EXISTS 语法参考为以下2个案例

搜索：exists

改为：

DECLARE

@ObjectID varchar2 (36 char);

@dept varchar2 (36 char);

BEGIN

if (not exists(select top 1 1 from MM_TAB_Favorites where UserID=UserID_ and InstanceID=InstanceID_)) then

INSERT INTO I_ITITems (ObjectID, dept) VALUES (@ObjectID, @dept);

else

UPDATE I_ITITems SET dept=@dept WHERE ObjectID=@ObjectID;

end if;

end;

改为：

DECLARE

BEGIN

if(@StandardLevel='公司级') then

select ObjectID,Name as DeptName,'部门级' category from OT_OrganizationUnit where ParentID='18f923a7-5a5e-426d-94ae-a55ad1a4b240' and Name like CONCAT('%', @Names,'%') order by SortKey ASC;

10、创建临时表及临时变量语法参考以下语法

搜索：table #

搜索：into #

搜索：declare

DECLARE

myQueryCode varchar2 (360 char);

BEGIN

create temp table tem (

ObjectID char(36) primary key

);

create temp table tem2 (

ObjectID char(36) primary key

);

insert into tem select @DeptID;

insert into tem2 select @DeptID;

WHILE (exists(select top 1 1 from tem2 a inner join OT_OrganizationUnit b on b.ParentID=a.ObjectID)) LOOP

insert into tem select b.ObjectID from tem2 a inner join OT_OrganizationUnit b on b.ParentID=a.ObjectID;

delete from tem2;

insert into tem2 select b.ObjectID from tem2 a inner join OT_OrganizationUnit b on b.ParentID=a.ObjectID;

END LOOP;

select myQueryCode=QueryCode from MM_Tab_Bylaw_Dictionary where ObjectID=@did;

select * from(

select

ROW_NUMBER() over(order by fi.PublishDate desc) as rowNUMBER

,fi.ID,fi.FileName ,fi.FinalFileCode,dept.Name as DeptName,fi.PublishDate

from MM_Tab_Bylaw_Files fi

inner join MM_Tab_Bylaw_Dictionary dic on dic.ObjectID=fi.DictionaryID

inner join OT_OrganizationUnit dept on dept.ObjectID=fi.DeptID

where exists(select top 1 1 from tem z where z.ObjectID=fi.DeptID)

and fi.FileState != 5 and dic.QueryCode like CONCAT(myQueryCode,'%')

and not exists(select top 1 1 from PlatFrom_InspectionAssessmentFinished x

inner join PlatFrom_ExamineStandardAndFileRelations y on y.ExamineStandardID=x.StandardID and y.FileID=fi.ID

where x.state=0 and x.IsConfirm=1 and x.StandardLevel='公司级' {0})

) t;

DROP TABLE tem;

DROP TABLE tem2;

end;

11、while循环从begin开始改为从LOOP开始

搜索：while

改为：

WHILE (exists(select top 1 1 from tem2 a inner join OT_OrganizationUnit b on b.ParentID=a.ObjectID)) LOOP

insert into tem select b.ObjectID from tem2 a inner join OT_OrganizationUnit b on b.ParentID=a.ObjectID;

delete from tem2;

insert into tem2 select b.ObjectID from tem2 a inner join OT_OrganizationUnit b on b.ParentID=a.ObjectID;

END LOOP;

12、参数化查询时，参数值不能直接使用枚举

搜索：.int

例如：parameters.Add(new Parameter("state", DbType.Int64, AssessmentStateEnum.Valid));

改为：parameters.Add(new Parameter("state", DbType.Int64, (int)AssessmentStateEnum.Valid));

13、Name会自动转小写，如需保留大写则加双引号

搜索：Name

例如：SELECT ObjectID,ParentID,Name,SortKey,IsFavorite FROM MM_Tab_Bylaw_Dictionary order by SortKey

改为：SELECT ObjectID,ParentID,"Name",SortKey,IsFavorite FROM MM_Tab_Bylaw_Dictionary order by SortKey
