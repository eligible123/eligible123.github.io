云枢列表导出数据失败问题处理

概述：
云枢列表导出报错是因系统缺少基础字体。在线解决方案:安装fontconfig并刷新字体缓存后重启webapi。离线方案:下载fontconfig的rpm包安装，

检查/usr/share/fonts日录，放入并解压truetype.zip，刷新

缓存后重启webapi。

报错信息：

问题原因或解决思路：
此错误和云枢没有关系，导出依赖系统自带字体服务，需保证操作系统或者镜像的基础环境包含基础字体。

解决方案
在线：

1.yum install fontconfig 2.fc-cache --force 重启webapi
