字段映射表数据同步技术方案

业务场景

在企业信息化系统中，需要将表 A 的数据根据字段映射规则同步到表 B，以实现数据迁移、转换或整合。字段映射规则定义了源表字段与目标表字段的对应关系，并支持不同数据类型的转换（如文本、日期、人员选择等）。同步过程需确保数据完整性、支持主表和子表处理，并记录详细日志以便于调试和审计。

核心需求

- 字段映射配置：根据配置表（FIELD_MAPPING_CONFIG 和 FIELD_MAPPING_DETAIL）获取源表和目标表的字段映射规则。 - 数据查询与转换：从源表查询数据，按照映射规则转换为目标表格式，支持多种数据类型（如文本、日期、人员、部门、附件等）。 - 主表与子表处理：支持主表数据和子表数据的同步，保持主子表关系（通过 refId 和 parentId）。 - 数据保存：将转换后的数据保存到目标表，设置状态为“已完成”（COMPLETED）。 - 异常与日志管理：记录详细日志，捕获异常并返回明确的错误信息。

实现思路

- 查询字段映射配置：

- 从 FIELD_MAPPING_CONFIG 表获取主配置信息（源表编码 sourceSchemaCode、目标表编码 targetSchemaCode、子表配置等）。 - 从 FIELD_MAPPING_DETAIL 表获取字段映射规则（源字段、目标字段、类型）。

  • 查询源表数据：

    • - 使用 SQL 查询源表（sourceSchemaCode）的所有数据。 - 若存在子表，查询子表配置和数据，并根据 refId 关联主表。

  • 数据转换：

    • - 对主表数据，遍历字段映射规则，调用特定类型转换方法（如文本、日期、人员等）。 - 对子表数据，基于主表 ID 映射（idMappingMap），生成子表数据并设置 parentId。

  • 数据保存：

    • - 将转换后的主表和子表数据封装为 BizObjectModel，保存到目标表。 - 使用原创建人 ID 或默认用户 ID 保存数据。

  • 性能优化：

    • - 使用批量查询和分区处理（如 Lists.partition）减少数据库交互。 - 缓存字段映射配置（detailBySourceCodeMap）以避免重复查询。

  • 异常与日志：

    • - 记录每条数据的处理过程（批次、ID、成功/失败）。 - 捕获异常并返回明确的错误信息（如配置缺失、数据为空）。

    实现步骤与代码

    1. 数据同步主逻辑

    负责整体同步流程，包括配置查询、数据处理和保存。

    说明：

    - 使用 BizObjectCreatedModel 获取字段映射配置。 - 通过 dataSyncHandler 处理数据转换，dataSyncForSave 保存数据。 - 记录同步开始和结束时间，计算耗时，便于性能监控。

    2. 数据保存

    将转换后的数据保存到目标表。

    说明：

    - 将数据转换为 BizObjectModel 列表，逐条保存到目标表。 - 检查创建人信息，若缺失则使用默认用户 ID。 - 捕获保存异常并记录日志，避免中断整体流程。

    3. 数据转换处理

    处理主表和子表数据的字段映射和类型转换。

    说明：

    - 查询源表数据和字段映射配置。 - 使用 detailBySourceCodeMap 缓存字段映射规则，优化性能。 - 调用 dataSyncForFieldHandler 处理主表和子表数据转换。

    4. 字段转换与子表处理

    实现字段映射和数据类型转换，支持主表和子表。

    说明：

    - 主表数据按批次（Constant.BATCH_SIZE）处理，生成新 ID 并存储映射关系（idMappingMap）。 - 子表数据通过 refId 关联主表 ID，设置 parentId 后保存。 - 使用缓存（detailBySourceCodeMap 和 childrenDetailBySourceCodeMap）优化字段映射查询。

    5. 字段类型转换

    根据字段类型执行特定转换逻辑。

    说明：

    - 根据字段类型（如“单行文本”、“日期”、“人员单选”）调用对应的转换方法。 - 支持空值检查和默认转换（convertString）。 - 示例转换方法（如 convertDate）支持多种日期格式的规范化。

    6. 示例类型转换 - 日期

    处理多种日期格式的转换。

    说明：

    - 支持多种日期格式（如 dd/MM/yyyy HH:mm:ss、yyyy/MM/dd HH:mm:ss）的规范化。 - 使用正则表达式处理特殊格式（如补零）。
