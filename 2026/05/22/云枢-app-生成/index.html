云枢 app 生成

背景 ： 云枢项目很多时候可能需要用到移动端app，这时候可能就需要打包给云枢移动端套壳

​

必要的环境：

android studio

jdk

android sdk

ndk

gradle

​

第一步：

下载android studio

第二步

根据需要安装对应版本的android sdk 。android studio 新建项目即可下载

第三步

打开文档提供的压缩包，打开这个项目根据readme的步骤全量打包即可

（ps：压缩包可以找我获取，语雀无会员无法上传）

第四步

安装打包生成的apk ，输入对应环境的移动端访问地址即可
