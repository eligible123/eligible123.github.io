js执行线程

如果你已经安装了Python 3，可以在命令行中进入你的项目目录，然后运行以下命令来启动一个简单的HTTP服务器：

python -m http.server

如果你已经安装了Node.js，可以使用npm安装http-server：

npm install -g http-server

然后在命令行中进入你的项目目录，运行以下命令：

http-server
