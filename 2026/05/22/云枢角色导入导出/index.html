云枢角色导入导出

场景：用 excel 对单个角色组下的角色导入导出。

实现方案

操作 excel 使用 poi。

模板导出

  • 因为导入导出的 excel 标头都是固定的，所以预先构建好导入导出 excel 模板，提供下载模板接口，不需要每次导出都使用代码从头构造模板。
  • 将设置好的 excel 模板文件放置在后端项目的 resource/templates/excel/ 目录下。
  • 根据条件查询到需要导出的数据列表。
  • 使用 excel 模板操作工具来将数据列表放置到对应的 excel 表格下标中。（excel 工具代码会在代码附录中给出）。

    • - 将文件流写入响应体，导出模板。 - 模板文件参考：

    数据导入

  • 通过文件流来获取操作 excel 的对象 workbook。
  • 获取开始行下标和结束行下标，循环处理导入的数据。

  • 调用保存方法批量插入 h_org_role 表、h_org_role_group 表中。
