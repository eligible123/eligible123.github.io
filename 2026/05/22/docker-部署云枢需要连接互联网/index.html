docker 部署云枢（需要连接互联网）

1、环境准备，创建/data 目录

不存在data 需创建

1、准备资源包，下载相应的资源包（http://packages.cloudpivot.cn/）。上传资源包到服务器

2、解压资源包

unzip cloudpivot_v.6.13.22_20230908112911.zip

如果出现如下情况，需要下载安装unzip

下载安装指令：yum install -y unzip zip

3、解压得到文件夹

4、进入docker-compose目录，vi .env 修改配置

5、配置修改，如果mysql不安装在这台服务器，需要去掉mysql配置

修改后

6、镜像配置修改 vim docker-compose.yml ，去掉不要的镜像

留下需要的镜像脚本

7、执行初始化安装脚本 ./init.sh 需要几分钟

8、查询docker容器 docker ps

9、docker安装mysql，修改docker-compose下的配置文件，只留mysql配置

10、执行脚本 ./init.sh 等安装成功

11、docker ps查看是否启动

12、登入mysql修改白名单

进入容器指令： docker exec -it h3_mysql /bin/bash

之后登录mysql：mysql -uroot -p

输入密码登录

修改权限白名单

选择cloudpivot : use cloudpivot;

顺便把登录授权入口修改好:如果 是以域名登录,把下面的ip 改成域名就可

13、配置webapi的 application-prod.yml

进入webapi文件指令：cd /data/cloudpivot/program/backEnd/webapi/

配置mysql账号密码，配置账号密码

14、重启h3_webapi

15、查看启动日志，是否启动成功，出现如下表示启动成功

16、访问ip,进入后管 账号密码 admin/123456

17、查看产品信息

18、组织配置，有三种组织维护方式

19、配置sftp
