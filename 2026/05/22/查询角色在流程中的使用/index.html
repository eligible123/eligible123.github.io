查询角色在流程中的使用

查询角色在流程中的使用

一、需求

目前各部门人员变动时，特别是合规或质控人员变动，需要查询用户参与了哪些流程包括审批节点和传阅节点，流程模板多节点也多，查起来需要耗费大量时间和精力，人工手动遍历也可能会有疏漏，故需参照之前H3定制的查询工作组和委托查询的功能，开发云枢的查询功能

二、思路

1.由于目前云枢的流程节点的在数据库中是以加密的方式存储，而H3的存储方式是明文存储，故云枢不能向H3那样通过数据库的方式直接进行查询。

2.查询某个模板的的审批节点的审批人的配置，需要查询出来模板的加密信息，然后在进行解密，最后才能查询审批人的配置。

3.查询角色在哪些流程中使用，由于需要查询的模板的数目很多，而且还要查询历史模板，所以先查询出来的模板，然后循环解密在解析审批人的方式，会导致查询效率很低，故采用将流程模板的审批人存储在redis缓存中，模板的审批人没有在缓存中的，再从数据库中获取。流程模板发布的时候将审批人缓存到redis中方。

    private void setWorkflowTemplateActivityPartticipant(String workflowCode,int workflowVersion,List> participants){ this.redisTemplate.opsForValue().set("cloudpivot:workflowtemplate:"+workflowCode+":"+workflowVersion,JSON.toJSONString(participants));}

    //从缓存中获取模板审批人private List getWorkflowTemplateActivityParticipant(String workflowCode,int workflowVersion){ Object str=redisTemplate.opsForValue().get("cloudpivot:workflowtemplate:"+workflowCode+":"+workflowVersion); if(str==null || StringUtil.isEmpty(str.toString())){ return null; } List res= JSON.parseArray(str.toString(),JSONObject.class); return res;}

    //解析角色审批人中的角色private int handelActivityParticipant(RoleModel roleModel,String workflowCode,String participant,String activityCode,String activityName,List> result,String workflowName,int workflowversion,int num,boolean isLatest){ Map map = new HashMap<>(); try { map.put("activityCode", activityCode); map.put("activityName", activityName); map.put("workflowCode", workflowCode); map.put("workflowName", workflowName); map.put("workflowVersion",workflowversion); map.put("isLatest",isLatest?"是":"否"); if (StringUtil.isEmpty(participant)) { return num; } //匹配以"[{"开始，以"}]"结束的字符串（表达式试图配置的角色） String match = "\\[\\{(.*?)\\}\\]"; Pattern pattern = Pattern.compile(match); Matcher matcher = pattern.matcher(participant); while (matcher.find()) { String unitReg = matcher.group(1); JSONObject json = JSONObject.parseObject("{" + unitReg + "}"); String id = json.getString("id"); String type = json.getString("type"); //2表示角色 if (type.equals("2")) { if (id.equals(roleModel.getId())) { num++; map.put("num",num); result.add(map); } } }

    //公式视图配置的角色 String match2 = "FindUserByRole\\(.*\\)"; Pattern pattern2 = Pattern.compile(match2); Matcher matcher2 = pattern2.matcher(participant); while (matcher2.find()) { String unitReg = matcher2.group(0); String str = unitReg.replace("\"", "").replace(")", ""); String[] strs = str.split(","); if (strs[1].equals(roleModel.getCode())) { num++; map.put("num",num); result.add(map); } } } catch (Exception ex){ log.info("解析角色异常:"+participant+workflowCode+workflowversion); log.info(ex.getMessage(),ex); } return num;}
